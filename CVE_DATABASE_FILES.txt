DEPENDENCIES REQUIRED:
----------------------

To build the CVE database module:

Ubuntu/Debian:
  sudo apt-get install libsqlite3-dev libssl-dev

RHEL/Fedora:
  sudo dnf install sqlite-devel openssl-devel

Arch Linux:
  sudo pacman -S sqlite openssl

BUILD INSTRUCTIONS:
-------------------

1. Install dependencies (see above)

2. Build FirmwareGuard:
   cd /home/zero/FirmwareGuard
   make clean
   make

3. Build test utility:
   gcc -Wall -Wextra -O2 -std=gnu11 -Iinclude -D_GNU_SOURCE \
       tools/test-cve-db.c \
       src/database/cve_db.c \
       src/cJSON.c \
       -lsqlite3 -lssl -lcrypto \
       -o tools/test-cve-db

4. Run tests:
   ./tools/test-cve-db
